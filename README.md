# IfThenDo

**IfThenDo** is a customizable calendar, to-do list, and reminder system designed for busy students whose schedules constantly change.

Instead of forcing users to plan everything at fixed times, IfThenDo adapts automatically using **if → then logic**.

Examples:
1) If free time opens up → show priority tasks
2) an event is canceled → reorganize schedule
3) If weather changes → suggest relevant activities


The goal is to make scheduling feel natural for how people actually think.


# Why I Built This

Traditional planning tools didn’t work well with how my brain organizes tasks. Most calendars assume your day is fixed, but real schedules change all the time.

IfThenDo is designed to:
- Adapt when your schedule changes
- Suggest what to do next
- Reduce stress from rigid planning


# Core Features

### Adaptive Scheduling
Tasks automatically reorganize when your availability changes.

### Conditional Task Logic
Create rules like:
IF free time > 30 minutes
THEN show homework tasks


### Tasks Between Events
The system detects gaps in your day and fills them with useful tasks.

### Flexible Deadlines
Tasks can be completed within ranges instead of exact times.

Example:
- Complete between Monday – Wednesday
- Do after practice
- Anytime this week

### Full Customization
Users can personalize:
- colors
- themes
- layouts
- checkbox styles
- categories
- dashboard view

### Smart Categories
Organize tasks by:
- School
- Work
- Personal
- Projects
- Activities

### Location-Based Tasks
Example:
- At school → assignments appear
- At home → chores appear

### Weather-Based Suggestions
Example:
- If snowing → shovel driveway
- If sunny → go biking

### Shared Planning
Collaborate with:
- friends
- group projects
- clubs
- teams

### Widgets and Mobile Support
Future goal:
- phone widgets
- quick add tasks
- daily overview
- mobile-friendly interface
